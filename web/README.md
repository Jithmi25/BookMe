# BookMe Web

This folder is reserved for the Next.js web application.

## Planned stack

- Next.js 14+ with TypeScript
- Tailwind CSS
- Firebase (Auth, Firestore, Storage, Cloud Functions)

## Next step (Day 1 scaffold)

Run from repository root:

```bash
npx create-next-app@latest web --typescript --tailwind --eslint
```

If this folder is not empty, run the command in a temporary folder and move generated files into `web/`.
